Projeto de Simulação de Forrageamento Inteligente por Cárcara

Utiliza de Algoritimos de Busca e Q-Learning

Main.py, roda o programa com a interface gráfica, escolhendo de 1-5 a forma que deseja rodar (descrito no topo da aba aberta).

Benchmark.py, roda mutiplas instancias documentando elas em um csv e printando a média delas no terminal.
